What you get

Mainnet behavior, no mainnet risk.

A sandbox isn't useful if its behavior diverges from production. Ours forks the canonical chain state at the block you specify and applies your transactions on top. Same contracts, same gas, same revert behavior. Just nobody loses real money.

Real chain state
Forked at block N. All deployed contracts, balances, allowances, oracle feeds, exactly as they are on mainnet at fork time.
Pre-funded test wallet
Each sandbox issues a wallet with 100 ETH worth of gas + 10K USDC. Enough to run any reasonable agent strategy end-to-end.
Snapshot + rewind
Take a snapshot before an experiment, run it, restore. Run twenty variants of your agent's logic against the same starting state.
Auto-cleanup
Sandboxes self-destruct after 1h idle. Or call DELETE /api/x402/sandbox/:id to release immediately. No orphaned compute.
