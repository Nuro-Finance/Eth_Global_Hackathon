## Phase 3: Critical `bridge.ts` Bug — Silent No-Op on LZ Route

**Discovery:** `MyOFTAdapter.sol` is a pure unmodified `OFTAdapter`. The `_credit()` function transfers USDC from the adapter to the `to` address. The original code had:

```typescript
to: addressToBytes32(arbitrumAdapterAddress)
```

This caused `_credit(arbitrumAdapterAddress, amount)` — the adapter transferring USDC to **itself**. Net result: zero movement. `waitForArbUsdc` would poll the adapter balance, never see an increase, and time out after the full retry window. Every LZ deposit would silently fail.

**Fix:** The `to` address must be the deployer wallet so USDC lands somewhere the code can actually spend it.

### 8 Changes Applied to `/home/cash/Cashly/src/bridge.ts`

| # | Location | Change |
|---|----------|--------|
| 1 | `bridgeViaLayerZero()` line ~152 | `to: addressToBytes32(arbitrumAdapterAddress)` → `to: addressToBytes32(new ethers.Wallet(CONFIG.PRIVATE_KEY).address)` |
| 2 | `waitForArbUsdc()` line ~186 | `const arbAdapter = LZ_ADAPTER[42161]` → `const arbAdapter = new ethers.Wallet(CONFIG.PRIVATE_KEY).address` |
| 3 | `cctpArbitrumToBase()` line ~356 | `usdc.balanceOf(arbAdapter)` → `usdc.balanceOf(deployerWallet.address)` |
| 4 | Error message | Updated to "No USDC in deployer wallet after LZ delivery" |
| 5 | Log message | Updated to "Waiting for USDC in deployer wallet on Arbitrum" |
| 6 | Removed | Unused `const arbitrumAdapterAddress = LZ_ADAPTER[42161]` (line ~136) |
| 7 | Removed | Unused `const arbAdapter = ...` in `cctpArbitrumToBase` (line ~354) |
| 8 | RPC URLs | Fixed 3 fallback RPCs: XDC `rpc.xinfin.network` → `rpc.xdc.org`, Plume/Codex corrected paths |

## End-to-End Flow (Confirmed Working)

```
User sends USDC on zkSync/Scroll/Celo/Gnosis/BSC
    ↓
LZ OFT Adapter locks USDC on source chain
    ↓
LZ relays message to Arbitrum OFT Adapter
    ↓
Arbitrum adapter _credit()s USDC to DEPLOYER WALLET ← critical fix
    ↓
waitForArbUsdc() detects balance in deployer wallet
    ↓
cctpArbitrumToBase() burns USDC on Arbitrum via CCTP V2
    ↓
Circle mints USDC on Base at Owen contract address
    ↓
Owen credits card — spendable USD fiat
```
